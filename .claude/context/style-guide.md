## I. Design System Foundation

### Color Palette

Our brand is red (#792420), blue (#4c9cc3), black (#000000) . Make the color graceful in visual.

### Typography Guidelines

- [ ] Display font: Rounded, playful sans-serif for headings
- [ ] Body font: Clean, highly readable sans-serif
- [ ] Clear size hierarchy from hero → headings → body
- [ ] Generous line height for readability

### Spacing & Layout

- [ ] Consistent spacing scale (8px base recommended)
- [ ] Generous whitespace throughout
- [ ] Soft, rounded corners — avoid sharp edges
- [ ] Warm, subtle shadows for depth
