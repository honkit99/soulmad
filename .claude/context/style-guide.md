## I. Design System Foundation

### Color Palette

Our brand identity is built on **RED and BLUE** as primary colors, with supporting neutrals.

**Primary Brand Colors:**

- Red: #792420 (primary actions, urgency, excitement)
- Red Light: #9a3b36
- Red Dark: #5a1b18
- Red Tint: #b34540
- Blue: #4c9cc3 (community, trust, secondary actions)
- Blue Light: #6db3d4
- Blue Dark: #3a7a9a
- Blue Tint: #7fb8d9
- Purple: #6b4c7c (red+blue blend for special elements)

**Neutrals:**

- Black: #000000 (text only, used sparingly)
- White: #ffffff
- Cream: #f5f0e8 (warm background)
- Gray Dark: #2a2a2a
- Gray: #333333
- Gray Light: #666666

**Color Usage Guidelines:**

- Red: ~40% of brand-critical elements (primary CTAs, section tags, highlights)
- Blue: ~35% of brand-critical elements (secondary CTAs, alternate section tags, trust elements)
- Black: ~5% (text, deep shadows only)
- Neutrals: ~20% (backgrounds, cards)

**Implementation:**

- Section tags alternate between red and blue throughout the page
- Stats bar and testimonials use red-blue gradient backgrounds
- Footer uses dark red (#5a1b18) with blue link accents
- Placeholder images use red/blue gradients instead of gray
- Secondary buttons are blue (#4c9cc3)
- Primary buttons remain red (#792420)

### Typography Guidelines

- [ ] Display font: Rounded, playful sans-serif for headings
- [ ] Body font: Clean, highly readable sans-serif
- [ ] Clear size hierarchy from hero → headings → body
- [ ] Generous line height for readability

### Spacing & Layout

- [ ] Consistent spacing scale (8px base recommended)
- [ ] Generous whitespace throughout
- [ ] Soft, rounded corners — avoid sharp edges
- [ ] Warm, subtle shadows for depth
