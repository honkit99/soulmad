# Board Game Shop Landing Page Design Principles

## A Joyful, Modern Showcase for Community & Play

---

## I. Core Design Philosophy

### Brand Essence: "Happiness Through Play"

- [ ] **Joy First:** Every design decision should evoke warmth, fun, and the excitement of game nights
- [ ] **Community Vibes:** Showcase the social, welcoming atmosphere of the shops. Style should use vintage american style.
- [ ] **Authenticity:** Use real photos of actual shops, games, and customers — avoid generic stock
- [ ] **Approachable Premium:** High-quality and modern, but never intimidating
- [ ] **Clarity Over Cleverness:** Visitors should find key info (shops, prices, location) within seconds

### Design Personality

- [ ] Playful but professional
- [ ] Cozy and inviting — like a café meets game library
- [ ] Clean and effortless to navigate

---

---

## III. Content Requirements

### Must-Have Information

- [ ] **Two Shops Differentiation:**
  - Each shops needs: name, description, location, hours, photos

- [ ] **Environment Showcase:** Photos of shop interiors and atmosphere

- [ ] **Games Display:** Featured games with relevant info

- [ ] **Pricing & Menu:** Clear rates, food/drinks if applicable

- [ ] **Social Media:** Links and/or feed integration

- [ ] **Contact & Location:** Addresses, hours, maps

### Content Tone

- [ ] Warm and welcoming
- [ ] Playful but clear
- [ ] Inclusive — everyone belongs

---

## IV. Interaction & Animation Principles

- [ ] Purposeful — every animation communicates something, use gsap and also gsap mcp for more ability on design
- [ ] Playful, not distracting — subtle and smooth
- [ ] Fast and responsive
- [ ] Respect prefers-reduced-motion

---

## V. Responsive Design

- [ ] **Primary:** Optimize for mobile and tablet
- [ ] **Secondary:** Fully functional on ipad and desktop too
- [ ] Graceful layout adaptation across breakpoints

---

## VI. Performance & Accessibility

### Performance

- [ ] Optimized images
- [ ] Fast initial load
- [ ] Smooth interactions

### Accessibility (WCAG AA)

- [ ] Sufficient color contrast
- [ ] Screen reader compatible
- [ ] Proper heading hierarchy

---

## VII. Visual Guidelines

### Photography

- [ ] Warm, natural lighting
- [ ] Authentic, candid moments
- [ ] Consistent color grading

### UI Elements

- [ ] Consistent icon set
- [ ] Friendly, rounded buttons
- [ ] Cards with soft shadows
- [ ] Pill-shaped badges for status/categories

---

## VIII. Do's and Don'ts

### ✅ DO

- Make the two shops clearly distinguishable
- Show authentic happiness and community
- Keep pricing and hours easily findable
- Use warm, inviting visual language

### ❌ DON'T

- Use generic stock photography
- Overwhelm with excessive animations
- Hide important information
- Use cold, corporate aesthetics

---

## IX. Page Structure

> **Research similar successful websites** (board game cafés, cozy retail, community spaces) to inform optimal section structure. Propose layout based on findings.
